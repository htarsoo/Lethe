## General info
Lethe is a lightweight pentest framework. It uncludes modules for Information Gathering, Vulnerability analysis, Explotation and more.
	
## Modules

### 1.Information Gathering: 
  *  WhoIS scan 
  *  Port scanner 
  *  Subdomain Scanner 
  *  Spider/Crawler
### 2.Vulnerability Analysis: 
  * SQLI scan 
  * SSL scan 

### 3.Exploiting: 
  * SQL Injection 
  * DOS
  	-Slowloris 
	-SYN flood 
### 4.Password:

		Hash Identifier
